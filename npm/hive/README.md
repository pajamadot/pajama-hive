# @pajamadot/hive

Pajama Hive CLI — Agent Orchestrator for AI coding agents (Claude Code, Codex CLI, and more).

## Install

```bash
npm install -g @pajamadot/hive
```

## Usage

```bash
# Authenticate
hive login

# Connect to server and start accepting tasks
hive connect

# Check status
hive status
```
